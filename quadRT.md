# PARSE
### handling error
***
1. `A`, `C`, `L` 이 포함되지 않은 경우 혹은 중복 되는 경우
   - `A` = 주변조명
   - `C` = 카메라
   - `L` = 빛
     
2. `figure`가 들어오지 않은 경우
   - 구
   - 원통
   - 평면
   - 2도 물체 : 원뿔 등
     
3. 수의 범위가 `INT_RANGE`를 벗어 난 경우
   
4. 영문자, 특수문자 포함
   
5. "`x`, `y`, `z`", "`r`, `g`, `b`" 값이 모두 들어오지 않은 경우
   - 모든 요소에 하나의 값
     
6. 각 요소들의 인자 갯수 불일치
